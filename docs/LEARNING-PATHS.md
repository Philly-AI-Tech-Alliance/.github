## Self-Assessment Guide

Before choosing a path, assess your current level:

### Technical Skills Assessment

**Beginner (0-1 year experience):**
- [ ] Little to no programming experience
- [ ] Unfamiliar with AI/ML concepts
- [ ] Want to learn what AI is and how it works
- [ ] Looking to start a career in AI

**Intermediate (1-3 years experience):**
- [ ] Comfortable with Python programming
- [ ] Understand basic ML concepts
- [ ] Have built simple models
- [ ] Want to deepen technical skills

**Advanced (3+ years experience):**
- [ ] Strong programming and ML background
- [ ] Have shipped ML models to production
- [ ] Understand deep learning fundamentals
- [ ] Want to specialize or stay current

### Learning Style Assessment

**Structured Learner:**
- Prefer step-by-step curricula
- Like clear learning objectives
- Enjoy comprehensive courses
→ Choose curriculum-based paths (Paths 1-4, 6)

**Project-Based Learner:**
- Learn best by building
- Prefer hands-on challenges
- Like figuring things out
→ Choose [Path 5: Hands-On AI Projects](#path-5-hands-on-ai-projects-track)

**Hybrid Learner:**
- Mix of both approaches
- Like theory + practice
→ Start with structured path, add projects along the way

### Time Commitment Assessment

**Full-Time (20+ hours/week):**
- Can complete beginner path in 4-6 weeks
- Intensive, fast-paced learning
- Regular daily practice

**Part-Time (10-15 hours/week):**
- Can complete beginner path in 8-12 weeks
- Balanced approach
- Recommended for working professionals

**Casual (5-10 hours/week):**
- Can complete beginner path in 3-6 months
- Steady, sustainable pace
- Fits alongside other commitments

**Self-Paced (varies):**
- Complete at your own pace
- No time pressure
- Learn when you have time

## Path 1: AI Fundamentals Track

**Target Audience**: Complete beginners to AI/ML, career changers, students exploring AI

### Overview

This path provides a comprehensive introduction to artificial intelligence and machine learning, assuming no prior AI knowledge. You'll learn fundamental concepts, build your first ML models, and gain confidence working with AI tools and technologies.

### Prerequisites

**Required:**
- Basic Python programming (variables, functions, loops)
- High school mathematics (algebra, basic statistics)
- Comfortable using a computer and terminal

**Optional but helpful:**
- Jupyter notebook experience
- Git/GitHub basics
- Basic data analysis experience

**Don't have prerequisites?** Start here:
- [Python for Beginners](https://www.python.org/about/gettingstarted/)
- [Git Handbook](https://guides.github.com/introduction/git-handbook/)

### Learning Objectives

By completing this path, you will be able to:
- Explain what AI and ML are and how they work
- Understand key ML concepts (supervised/unsupervised learning, training, evaluation)
- Build and train classification and regression models
- Use popular ML libraries (scikit-learn, pandas, matplotlib)
- Evaluate model performance and interpret results
- Apply AI tools to real-world problems
- Identify when ML is (and isn't) the right solution

### Path Structure

#### Phase 1: Foundations (40 hours)

**Repository**: [`ai-fundamentals`](https://github.com/Philly-AI-Tech-Alliance/ai-fundamentals)

**Modules:**
1. **Introduction to AI** (4 hours)
   - What is AI? History and evolution
   - Types of AI: Narrow vs. General
   - Real-world applications
   - AI capabilities and limitations

2. **Machine Learning Basics** (8 hours)
   - Supervised vs. unsupervised learning
   - Training, validation, testing
   - Feature engineering
   - Model selection

3. **Neural Networks Introduction** (10 hours)
   - Biological inspiration
   - Perceptrons and activation functions
   - Backpropagation basics
   - Introduction to deep learning

4. **Practical AI Tools** (8 hours)
   - Using ChatGPT and LLMs effectively
   - GitHub Copilot for coding
   - No-code/low-code ML platforms
   - AI tool ecosystem overview

5. **Hands-On Project** (10 hours)
   - Build a complete classification system
   - From data collection to deployment
   - Model evaluation and iteration
   - Documentation and presentation

#### Phase 2: Application (20 hours)

**Workshop Series** (attend 3-4 workshops):
- Introduction to Prompt Engineering
- Building Your First Chatbot
- Computer Vision Basics
- Natural Language Processing Intro

**Project Work**:
- Complete one capstone project
- Document in GitHub repository
- Present to community
- Get feedback from mentors

#### Phase 3: Community Integration (ongoing)

**Join Study Group**:
- Weekly sync meetings
- Peer learning and support
- Accountability partners

**Contribute Back**:
- Improve course materials
- Answer questions in Slack
- Share your learning journey
- Help next cohort

### Time Estimate

- **Total Hours**: 60-80 hours
- **Recommended Pace**: 8-10 weeks at 6-8 hours/week
- **Intensive Option**: 4-6 weeks at 12-15 hours/week
- **Relaxed Option**: 3-6 months at 3-5 hours/week

### Next Steps After Completion

**Intermediate Paths:**
- [Path 2: Machine Learning Practitioner](#path-2-machine-learning-practitioner-track) - Deepen technical ML skills
- [Path 3: Deep Learning Specialist](#path-3-deep-learning-specialist-track) - Focus on neural networks

**Complementary Learning:**
- [Path 6: Responsible AI Development](#path-6-responsible-ai-development-track) - AI ethics and best practices

**Community Involvement:**
- Join a project team
- Mentor new learners
- Contribute to open source projects

### Success Metrics

You'll know you're ready to move on when you can:
- [ ] Explain ML concepts to a non-technical friend
- [ ] Build a classification model from scratch
- [ ] Evaluate and compare different models
- [ ] Use scikit-learn for common ML tasks
- [ ] Debug basic ML issues
- [ ] Complete the capstone project successfully

## Path 2: Machine Learning Practitioner Track

**Target Audience**: Developers with Python experience, data analysts transitioning to ML, graduates wanting practical skills

### Overview

This intermediate path takes you deep into machine learning algorithms, best practices, and production-ready ML. You'll learn to build robust ML systems, optimize models, and deploy to production.

### Prerequisites

**Required:**
- Completed [Path 1: AI Fundamentals](#path-1-ai-fundamentals-track) or equivalent
- Strong Python programming skills
- Linear algebra and statistics basics
- Experience with pandas and numpy

**Validation Quiz**: Take self-assessment to confirm readiness

### Learning Objectives

- Implement ML algorithms from scratch
- Master feature engineering and selection
- Optimize hyperparameters systematically
- Build production-ready ML pipelines
- Deploy and monitor models
- Handle imbalanced datasets and real-world messiness
- Understand model interpretability

### Path Structure

#### Phase 1: Algorithm Deep Dive (50 hours)

**Repository**: `intermediate-ml` (coming soon)

**Modules:**
1. Linear Models and Regularization
2. Tree-Based Methods (Decision Trees, Random Forests, Gradient Boosting)
3. Support Vector Machines
4. Ensemble Methods
5. Dimensionality Reduction
6. Clustering Algorithms

#### Phase 2: Production ML (40 hours)

**Repository**: `mlops-fundamentals` (coming soon)

**Topics:**
- ML pipelines and automation
- Model versioning and tracking (MLflow)
- Deployment strategies (API, batch, streaming)
- Monitoring and maintenance
- A/B testing
- CI/CD for ML

#### Phase 3: Specialization Project (30 hours)

Choose one domain:
- Computer Vision
- Natural Language Processing
- Time Series Forecasting
- Recommender Systems

Build and deploy a complete system.

### Time Estimate

- **Total Hours**: 120-150 hours
- **Recommended Pace**: 12-16 weeks at 8-10 hours/week
- **Intensive Option**: 8-10 weeks at 12-15 hours/week

### Next Steps

- [Path 3: Deep Learning Specialist](#path-3-deep-learning-specialist-track)
- Specialize in domain (NLP, Computer Vision, etc.)
- Lead workshops and mentor others
- Contribute to open source ML projects

## Path 3: Deep Learning Specialist Track

**Target Audience**: ML practitioners wanting to master neural networks and deep learning

### Overview

Master neural networks, from fundamentals to state-of-the-art architectures. Learn to build, train, and deploy deep learning models for vision, language, and other domains.

### Prerequisites

**Required:**
- Completed [Path 2: Machine Learning Practitioner](#path-2-machine-learning-practitioner-track) or equivalent
- Strong ML foundations
- Calculus and linear algebra
- GPU computing basics

### Learning Objectives

- Understand deep learning theory
- Build neural networks with PyTorch/TensorFlow
- Master CNNs for computer vision
- Master transformers for NLP
- Fine-tune pre-trained models
- Optimize training (learning rates, batch sizes, etc.)
- Debug gradient issues and training problems

### Path Structure

#### Phase 1: Deep Learning Foundations (60 hours)

**Repository**: `deep-learning-practitioner` (coming soon)

**Modules:**
1. Neural Network Theory
2. Backpropagation and Optimization
3. Convolutional Neural Networks
4. Recurrent Networks and LSTMs
5. Attention Mechanisms
6. Transformers Architecture

#### Phase 2: Advanced Topics (50 hours)

**Topics:**
- Transfer learning and fine-tuning
- Generative models (GANs, VAEs, Diffusion)
- Reinforcement learning basics
- Model compression and optimization
- Distributed training

#### Phase 3: Specialization (40 hours)

Deep dive into one area:
- Computer Vision (object detection, segmentation, etc.)
- Large Language Models
- Generative AI
- Multi-modal models

### Time Estimate

- **Total Hours**: 150-180 hours
- **Recommended Pace**: 16-20 weeks at 8-10 hours/week

### Next Steps

- Research contributions
- Kaggle competitions
- Open source model development
- Speaking at conferences

## Path 5: Hands-On AI Projects Track

**Target Audience**: Project-based learners, builders, those wanting portfolio projects

### Overview

Learn by building! This path guides you through progressively complex AI projects, from simple scripts to production systems. Perfect for those who learn best by doing.

### Prerequisites

**Flexible based on project:**
- Beginner projects: Basic Python
- Intermediate projects: [Path 1](#path-1-ai-fundamentals-track) completion
- Advanced projects: [Path 2](#path-2-machine-learning-practitioner-track) completion

### Learning Approach

Instead of structured curriculum, you'll:
1. Choose a project from the project library
2. Follow project tutorial or build independently
3. Get mentorship and code reviews
4. Share and document your work
5. Move to next project

### Project Library

**Beginner Projects** (10-20 hours each):
- Sentiment Analysis for Movie Reviews
- Image Classifier for Objects
- Chatbot with Rule-Based Logic
- Recommendation System (simple)

**Intermediate Projects** (30-50 hours each):
- Fine-Tune a Language Model
- Object Detection System
- Time Series Forecasting App
- Deploy ML Model as API

**Advanced Projects** (60-100 hours each):
- Build a Search Engine
- Create a Generative AI Application
- Multi-Modal Learning System
- Production ML Platform

### Structure

**For each project:**
1. **Plan** (10% of time): Requirements, architecture, learning goals
2. **Build** (60% of time): Implementation, iteration, debugging
3. **Polish** (20% of time): Documentation, testing, code quality
4. **Share** (10% of time): Demo, write-up, community presentation

### Time Estimate

- **Highly variable**: Depends on projects chosen
- **Typical**: Complete 3-4 beginner projects, then 1-2 intermediate
- **Duration**: 3-6 months at 10-15 hours/week

### Next Steps

- Build portfolio for job applications
- Turn projects into tutorials
- Mentor others on similar projects
- Start your own startup

## Path 6: Responsible AI Development Track

**Target Audience**: All AI practitioners, ethicists, policy makers, anyone building AI systems

### Overview

Learn to develop AI systems ethically and responsibly. Understand bias, fairness, privacy, transparency, and societal impact. This path complements any technical track.

### Prerequisites

**Required:**
- Basic understanding of AI/ML
- Interest in ethics and societal impact

**Recommended:**
- [Path 1: AI Fundamentals](#path-1-ai-fundamentals-track) or equivalent
- Some ML development experience

### Learning Objectives

- Identify and mitigate bias in AI systems
- Ensure fairness across demographic groups
- Implement privacy-preserving techniques
- Make AI systems transparent and explainable
- Understand AI policy and regulation
- Design inclusive AI products

### Path Structure

#### Phase 1: AI Ethics Foundations (25 hours)

**Repository**: `ai-ethics-practice` (coming soon)

**Modules:**
1. Introduction to AI Ethics
2. Bias and Fairness
3. Privacy and Data Protection
4. Transparency and Explainability
5. Accountability and Governance
6. Environmental Impact

#### Phase 2: Practical Implementation (20 hours)

**Workshops:**
- Bias Detection and Mitigation
- Privacy-Preserving ML
- Explainable AI Techniques
- Inclusive Design Practices

#### Phase 3: Case Studies and Application (15 hours)

**Activities:**
- Audit existing AI systems
- Design ethical AI framework
- Contribute to policy discussions
- Lead ethical review process

### Time Estimate

- **Total Hours**: 60-70 hours

### Next Steps

## Frequently Asked Questions

### How long does a path take?

Varies by path and your pace:
- **Beginner paths**: 2-6 months part-time
- **Intermediate paths**: 3-6 months part-time
- **Advanced paths**: 4-8 months part-time
- **Self-paced**: As long as you need!

### Can I switch paths?

Absolutely! Many learners:
- Start with structured path, switch to projects
- Explore multiple paths simultaneously
- Change paths as interests evolve

### Do I need to complete everything in order?

**Recommended but not required**. Some flexibility:
- Skip modules you already know (but review to confirm)
- Jump to projects if you're confident
- Take workshops in any order

**However**: Prerequisites exist for a reason. Struggling usually means you need more foundation.

### What if I get stuck?

Multiple support channels:
- **Slack**: #help channel for questions
- **GitHub Discussions**: Ask in Learning Together
- **Office Hours**: Weekly Q&A sessions
- **Mentors**: 1-on-1 help available
- **Study Groups**: Peer support

### Can I learn with others?

Yes! We encourage it:
- Form study groups
- Find learning partners
- Join workshops together
- Collaborate on projects

### How much does this cost?

**All content is free and open source!**

Optional costs:
- Cloud computing for large models ($5-50/month)

- Conference tickets (optional)
